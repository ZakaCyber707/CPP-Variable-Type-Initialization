Variable Type Initialization & Input


📊 Logic Overview
This project illustrates the fundamental lifecycle of variables in a C++ environment:
Declaration & Initialization: Variables are declared with specific types (int, float, double) and given initial baseline values.
Dynamic Update: The program uses cin to allow users to overwrite the initialized values during runtime.
Type-Specific Output: Each result is labeled and displayed to verify how the compiler stores and represents different numerical precisions.


🛠️ Features
Multi-Type Handling: Covers the three most common numeric types: int (integers), float (single-precision), and double (double-precision).
Interactive Console: Provides a clear step-by-step prompt for the user to enter data and see immediate results.


🚀 Execution & Environment Guide
To run this project, you will need a C++ compiler or a dedicated IDE.

1. Recommended IDEs
 Dev-C++
 Visual Studio Code
 Online Compilers: Use OnlineGDB or Programiz for instant browser-based execution.

2. Simple Execution Procedure
(Step 1): Setup the Project
Create a file named Variable_Initialization_Demo.cpp.
Copy and paste the source code into your editor.

(Step 2): Compile the Code
In Dev-C++: Press F11 to Compile and Run.
In Terminal: Use g++ Variable_Initialization_Demo.cpp -o VarDemo followed by ./VarDemo.

(Step 3): Observe the Results
Follow the prompts to enter different numbers and observe how the program stores and outputs each specific type.
